# 🏏 IPL 2022 Data Analysis

A data analysis project exploring the **IPL 2022** season using **Python**, **Pandas**, **Matplotlib**, and **Seaborn**. The project focuses on extracting meaningful insights from match data through exploratory data analysis (EDA) and data visualization.

---

## 📌 Project Overview

The objective of this project is to analyze IPL 2022 match data and answer important questions related to team performance, player achievements, toss decisions, and match outcomes.

---

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📂 Dataset

* IPL 2022 Match Dataset
* CSV Format

---

## 📊 Analysis Performed

This project includes analysis on:

* Which team won the most matches?
* Which team won the most tosses?
* Toss winner vs Match winner analysis
* Toss decision distribution
* Matches won by runs vs wickets
* Player of the Match awards
* Highest run scorer
* Best bowling figures
* Venue-wise match analysis
* Highest winning margin
* Highest individual score
* Which team has highest score in ipl 2022

---

## 📈 Visualizations

The project uses **Matplotlib** and **Seaborn** to create clear and informative visualizations for exploring IPL 2022 data.

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/Himanshupandey2005/ipl-2022-Data-Analysis.git
```

2. Move into the project folder

```bash
cd ipl-2022-Data-Analysis
```

3. Install the required libraries

```bash
pip install pandas matplotlib seaborn
```

4. Open the notebook

```bash
jupyter notebook index.ipynb
```
